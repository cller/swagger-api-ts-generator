
# 安装方法

```sh
  npm install swagger-api-ts-generator --g
```


# 使用方法
```
Usage: swagger-api-ts-generator [options] <uri>

Options:
  -V, --version                output the version number
  -o, --output [dir], default  Api output dir, default: "./api"
  -h, --help                   output usage information
```
